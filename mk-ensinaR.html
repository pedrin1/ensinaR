Uma ltima coisa interessante a ser feita, anlisar a taxa de letalidade da COVID-19. Para fazer isso precisamos calcular essa taxa e separar com as variveis que queremos exibir no grfico.

Para fazer criamos um DataFrame chamado casos que vai agrupar as variveis RacaCor e Sexo e fazer a contagem de casos confirmados. Depois, criamos um otro data frame com as mesmas variveis, porm esse vai contar o nmero de mortes pela COVID-19. Por ltimo, o DataFrame chamado letalidade foi responsvel por calcular a taxa de letalidade da COVID-19. Vale ressaltar que a taxa de letalidade a proporo entre o nmero de bitos por uma doena em relao ao nmero de indivduos acometidos por essa doena.

casos <- dados %>%
  group_by(RacaCor,Sexo) %>%
  summarise(n_casos = n())
mortes <- dados %>% 
  filter(Evolucao ==  "bito pelo COVID-19") %>%
  group_by(RacaCor,Sexo) %>%
  summarise(n_mortes = n())
letalidade <- inner_join(casos, mortes) %>%
  mutate(taxa_l =   ifelse(n_casos == 0, 0, 
                               round(100 * n_mortes / n_casos, 1)))

Certo, por ltimo iremos plotar o grfico

ggplot(letalidade) + geom_col(aes(x= RacaCor, y=taxa_l, fill= Sexo),position = "dodge") + 
  labs( x="Pessoas",
        y="Letalidade(%)",
        title="Taxa de letalidade da COVID-19 no ES") + 
  scale_fill_manual(values = c("yellow","purple")) + theme_bw()

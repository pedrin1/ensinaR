Como queremos fazer as anlises baseada nos casos confirmados, podemos comear aplicando um filtro para selecionarmos apenas os casos confirmados. Depois disso, usaremos um for para mudarmos os nomes da varivel Sexo e por ltimo, criamos uma nova varivel chama Idade que vai receber apenas os dois primeiros dgitos da varavel IdadeNaNotificacao.
A planilha original possui mais de 400mil observaes e para no puxar muita memria do computador iremos usar o cdigo dados <- dados[sample(1:dim(dados)[1], 20000), ] que ira restingrir a base de dados para que o computador consiga os cdigos de maneira mais rpida

dados <- read.csv("C:/Users/pedro/Desktop/MICRODADOS.csv", sep = ";")
dados <- filter(dados, Classificacao=="Confirmados")

dados <- dados %>% 
  mutate(DataNotificacao = as.Date(DataDiagnostico))
casos_t <- dados %>%
  group_by(Sexo, DataNotificacao) %>%
  summarise(n_casos = n()) %>%
  tidyr::complete(tidyr::nesting(Sexo), 
                  DataNotificacao = seq.Date(min(dados$DataNotificacao),
                                             max(dados$DataNotificacao), 
                                             by = "day"), 
                  fill = list(n_casos = 0)) %>%
  mutate(casos_acumulados = cumsum(n_casos))
ggplot(casos_t) + 
  theme_minimal() + 
  aes(x = DataNotificacao, y = casos_acumulados, color = Sexo) +
  geom_line() +
  scale_color_viridis_d() + 
  labs(y = "Nmero de casos acumulados", 
       x = "Data Diagnstico")

Observando o grfico de casos acumulados separado por sexo, podemos ver de forma mais clara o que foi dito antes, o sexo femino tem uma frequncia consideravelmente maior em relao ao masculino e foi maior em todo o intervalo de tempo analisado.

Uma ltima coisa interessante a ser feita, anlisar a taxa de letalidade da COVID-19. Para fazer isso precisamos calcular essa taxa e separar com as variveis que queremos exibir no grfico.

Primeiramente, criaremos uma nova varivel, onde essa ira receber o valor 1 (string) se a varivel Evolucao for igual a bito pelo COVID-19 e se for diferente ira receber o valor 0

attach(dados)
prop <- rep(nrow(dados))
for (i in 1:nrow(dados)){
  if(Evolucao[i]=="bito pelo COVID-19"){prop[i] <- "1"
  }else{prop[i] <- "0"}
}
dados$prop <- prop

Pronto, o prximo passo criar um DataFrame que ira receber as variveis que queremos analisar, nesse caso, Sexo, RacaCor e prop. Aqui j iremos calcular a taxa de letalidade.

taxa <- dados %>% 
  group_by(Sexo,prop,RacaCor) %>% 
  summarise(n=n()) %>% 
  filter(prop =="1") %>% 
  mutate(indice_mortalidade = (n/length(dados$prop))*100 )

Certo, por ltimo iremos plotar o grfico

ggplot(taxa) + geom_col(aes(x= RacaCor, y=indice_mortalidade, fill= Sexo),position = "dodge") +

Introduo

Diante da recente pandemia do novo coronavirus, ressalta-se a importncia da discusso de polticas pblicas para diferentes os grupos raciais da populao brasileira. Aqui no Esprito Santo, foi criado o Observatrio Capixaba da COVID-19 nas Favelas (OCOFA), com pesquisadores de diversas reas: epidemiologia, estatstica, cincias sociais, servio social, geografia, design e comunicao. Em especial, o objetivo deste observatrio produzir dados sobre os impactos sociais da pandemia nas favelas e na populao negra.

Com essa temtica em mente, nesse post vamos discutir como utilizar dados do Painel da Covid-19 do ES para mostrar como podemos utilizar essas informaes para produzir algumas anlises importantes nesse momento.

Por ltimo, uma ltima anlise interessante a ser feita, calcular a taxa de letalidade da COVID-19. Para fazer isso precisamos calcular esse ndice, podendo separar essa informao considerando algumas variveis de interesse.

Para fazer essa tarefa criamos um data.frame chamado casos agrupando as variveis RacaCor e Sexo, fazendo a contagem de casos confirmados. Depois, criamos um outro data.frame com as mesmas variveis, porm esse objeto vai contar o nmero de mortes pela COVID-19. Por ltimo, utilizamos a funo inner.join para gerar o data.frame chamado letalidade, onde calculamos a taxa de letalidade da COVID-19. Vale ressaltar que a taxa de letalidade a proporo entre o nmero de bitos por uma doena em relao ao nmero de indivduos acometidos por essa doena.

casos <- dados %>%
  group_by(RacaCor, Sexo) %>%
  summarise(n_casos = n())

mortes <- dados %>%
  filter(Evolucao ==  "bito pelo COVID-19") %>%
  group_by(RacaCor, Sexo) %>%
  summarise(n_mortes = n()) %>% 
  tidyr::complete(RacaCor, Sexo,
                  fill = list( n_mortes = 0))

letalidade <- inner_join(casos, mortes) %>%
  mutate(taxa_l =   ifelse(n_casos == 0, 0,
                           round(100 * n_mortes / n_casos, 1)))

Por ltimo podemos fazer o grfico dessa informao:

ggplot(letalidade) + 
  geom_col(aes(x = RacaCor, y = taxa_l, fill = Sexo), position = "dodge") +
  labs(x = "Pessoas",
       y = "Letalidade (%)",
       title = "Taxa de letalidade da COVID-19 no ES") +
  scale_fill_viridis_d() +
  theme_bw()

Ao observamos o grfico, notamos que a taxa de letalidade do sexo masculino sempre maior em relao ao sexo feminino em todas as raas. possvel notar tambm que para raa Indgena s observamos valores para o sexo Masculino. As maiores taxas para ambos os sexos foi observada na raa Amarela.
